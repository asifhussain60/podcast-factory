# Pronunciation Projection — Canonical Format for 03-pronunciation.md

The per-source pronunciation guide is a **projection** of the master lexicon at `JOURNAL_DIR/reference/translations-glossary.md`, filtered to only the terms that actually appear in this source. It is regenerated every run, never hand-edited.

## File header

```
# Pronunciation Guide — [Source Title]

Generated by /podcast skill on [YYYY-MM-DD] from terms detected in this source.
The master lexicon lives at `reference/translations-glossary.md`. To propose
new terms or corrections, see `06-library-proposals.md`.

This guide uses English phonetic transcription. Non-Latin script does not
appear in podcast-ready output. For the original-script form of any term,
consult the master lexicon.
```

## Body format

Group by language/script of origin. Within each group, alphabetize by phonetic spelling.

```
## Arabic-origin terms

| Phonetic | Meaning | First appearance in source |
|---|---|---|
| Allaah | God | Section 1, line 14 |
| Imaam | leader, guide | Section 1, line 23 |
| Sharee-ah | revealed law / outward path | Section 2, line 7 |
| Wilaayah | authority, guardianship | Section 3, line 41 |

## Hebrew-origin terms

| Phonetic | Meaning | First appearance in source |
|---|---|---|
| ...

## [Other script families as detected]
```

## Devotional phrases

Phrases (rather than single terms) get their own section at the end:

```
## Devotional phrases

| Phonetic | Meaning |
|---|---|
| Innaa lil-laahi wa innaa ilayhi raaji-oon | Indeed we belong to God, and to Him we return |
```

## What goes in this file

- Every non-Latin-origin term that appears in the source AND has a master-lexicon entry.
- Every new term detected in the source that does not yet have a master-lexicon entry (these also appear in `06-library-proposals.md` as proposed additions).

## What does NOT go in this file

- Terms that exist in the master lexicon but do not appear in this source.
- Terms that the source uses but that are already common English (e.g., "imam" as it appears in standard dictionaries — but use the master-lexicon entry to decide).
- Latin-script foreign loanwords (e.g., "ex nihilo") unless the master lexicon includes them.

## Provenance

Each entry's "First appearance in source" is a section reference (e.g., "Section 1, line 14"). This lets the user trace where a term comes from and verify the transcription against the original context.

## Hand-editing forbidden

This file is regenerated on every podcast run for this source. Any hand-edits will be overwritten. To change a term's transcription or meaning, propose the change against the master lexicon via `06-library-proposals.md`, then run `/podcast apply <slug>`.
