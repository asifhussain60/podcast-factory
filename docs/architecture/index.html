How the writing rooms actually work

Two skills (one for the memoir, one for the podcast) share a single brain for Arabic pronunciation. The nine views below explain the moving parts in plain language — pick a starting point. 2026-05-17: the podcast architecture pivoted to v3.5 after a 5-transcript empirical audit of Ayyuhal Walad. Phonetic guidance now lives in the customize prompt as imperative directives, never inline in the chapter. 2026-05-18 additions: conversation choreography (Driver vs Color roles, one cross-introduction beat, no mid-sentence interjections); R-NOMODERNIZE softened with an explicit "DO use modern-life practical analogies" paragraph; scaffold_book.py stamps a canonical BOOK_DIR layout in one shot; per-chapter chapter-contracts/<slug>.yml as the deterministic I/O surface; Category Q challenger checks for cross-book bleed; a debate episode format alongside deep_dive. The Arabic TTS protocol and the consolidated skill ROADMAP are now tracked under content/podcast/.skill/ and loaded by every producer and challenger run (cold-start items 17–18 of 18). A portable ChatGPT system prompt (in _workspace/) lets users drive Phase 0a–0e without running the repo skill.
