# Journal Podcast Source Processing Package

This package contains one markdown file per source file found in the uploaded archive, plus journal-app implementation instructions and OCR/translation tool recommendations.

Important limitation: the package does not contain complete raw English translations of every Arabic PDF. Most PDFs are scanned Arabic books with no usable embedded text layer. A reliable raw translation requires a paid OCR/translation API or the open-source OCR pipeline described in the instructions.
