# Pronunciation — Kunooz al-Hikmah (book-level overrides)

**Authority order:** `content/_shared/arabic/03-arabic-english-manifest.md`
wins for any term that appears there. This file holds book-specific entries
only and additive overrides. Contradictions with the manifest are defects to
fix here, not in the manifest.

## Title + author

| Term | Phonetic | Notes |
|---|---|---|
| Kunooz al-Hikmah | **koo-nooz al-HIK-mah** | The shortform book title. Plural of *kanz* (treasure). |
| Kunooz al-Hikmah al-Mazboora li-Arbaab al-Ismah | **koo-nooz al-HIK-mah al-maz-BOO-rah li-ar-BAAB al-IS-mah** | Full canonical title. Use on first mention in any chapter, shortform thereafter. |
| Ali bin Hibatullah al-Makrami | **A-lee bin hi-ba-TUL-lah al-MAK-ra-mee** | The author. *Sayyidina* honorific when speaker uses it. NEVER "Hibah Allah" or "Makrami" alone on first mention. |
| Hibatullah | **hi-ba-TUL-lah** | Father's name. *Hibah-Allah* literally "gift of Allah" — preserved as single compound. |

## Tayyibi-Bohra technical vocabulary

| Term | Phonetic | Notes |
|---|---|---|
| Dai | **DAA-ee** | Singular preacher/missionary. Two syllables; NOT "die". |
| Du'at | **doo-AAT** | Plural of Dai. |
| Hujjat | **HUJ-jat** | The "proof" rank below the Imam. |
| Hudood | **hoo-DOOD** | The hierarchical ranks; plural of *hadd*. |
| Wilayat | **wi-LAA-yat** | Spiritual authority/guardianship; NOT "vilayet" (Turkish admin). |
| Ismah | **IS-mah** | Infallibility. |
| Arbaab al-Ismah | **ar-BAAB al-IS-mah** | "Lords of Infallibility" — the Imams. |
| Asraar | **as-RAAR** | Secrets; plural of *sirr*. |
| Anwaar | **an-WAAR** | Lights; plural of *noor*. |
| Riyas | **ri-YAAS** | Subtle spiritual emanations (technical Bohra usage). |
| Nawasid-e-zilliyah | **na-WAA-sid-e ZIL-lee-yah** | Shadow-correlates of riyas; technical compound. |
| Adam-e-Aal-e-Aba | **AA-dam-e AAL-e a-BAA** | "Adam of the Family of the Cloak" — title of Imam Ali Zain-ul-Abideen. NEVER "Adam Allah" or letter-spelled. |
| Naasut / Malakut / Jabarut | **naa-SOOT / ma-la-KOOT / ja-ba-ROOT** | The three realms. |
| Nasur-e-Khaas | **NAA-soor-e KHAAS** | The "special manifestation" within Naasut; gradation series the speaker uses (Adna / Awsat / Khaas / Tabi / Zilli). |
